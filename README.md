# Clinic-paitent-visit-system-in-Java
System to enter data for patients, clinics, and visits with cmd inside the compiler .
using jdk17 or jre21 .
this is a project given to Birzeit university students who took the course comp2310 in the 2025 fall semester.
given with the java files the pdf's for the system uml and the project description .
suggested to use eclips 2022 or newer eclips IDE versions.
in this project data isnt saved into a files but rather inside ArrayLists while the compiler ran the code.
